# Graph

Interactive **entity relationship** explorer.

## Displays

- Nodes (entity type + label)
- Edges (relationship type)
- Edge confidence
- Source reference on hover
- Filters: type, relationship, min confidence, time range

## Backend

`graph-service` — [Backend services](/docs/platform/backend-services).

## Reference UX

[Maltego](https://www.maltego.com/transform-hub/) — Transform Hub connects data sources; transform docs describe mapping inbound entities to API/database results. Useful mental model for plugin architecture.

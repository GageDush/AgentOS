# Map

Geospatial layer for **location candidates** and evidence pins.

## Displays

- Base map (OSM tiles)
- Candidate locations ranked by confidence
- Evidence pins linked to [evidence cards](/docs/ui/evidence-card)
- Feature overlays from [Overpass](/docs/adapters/maps) queries
- Optional distance measurement tool
- Toggle source layers on/off

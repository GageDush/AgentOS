# Entity card

Compact summary of one normalized entity.

## Displays

- Entity type icon + label
- Canonical value
- First seen / last seen
- [Confidence](/docs/components/confidence-scoring) badge
- Source count
- Tags
- Quick actions: enrich, add to graph, link evidence

## Data source

`entities` table — [Data model](/docs/platform/data-model).

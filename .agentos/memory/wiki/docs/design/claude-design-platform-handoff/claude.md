# Claude Design Handoff

Start with `design.md`.

Use `prompt.md` when generating or rebuilding the site.

Use these folders as supporting evidence:

- `screenshots/` - desktop, full-page, and mobile captures.
- `assets/` - retrieved images, SVGs, and video files.
- `derived/` - extracted inline SVG and motion-reference notes.
- `pages/` - rendered HTML snapshots.
- `raw/` - original gallery and manifest JSON.

Recommended Claude workflow:

1. Attach or reference this folder.
2. Read `design.md`.
3. Use `prompt.md` as the build prompt.
4. Pull visual references from `screenshots/` and `assets/`.
5. Use `raw/gallery.json` only when exact captured metadata is needed.

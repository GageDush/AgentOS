# Claude Design Handoff

Start with `design.md`, then read `wireframe.md`.

This folder is only for the AgentOS dashboard/base site design.

Use:

- `design.md` for product direction and naming tone.
- `wireframe.md` for layout and completion scope.
- `assets/` for scraped website asset references.
- `screenshots/` for visual rhythm and responsive layout inspiration.
- `raw/` only when metadata is needed.

Do not turn the dashboard into a military, spy, or secret-ops interface.

# AgentOS Forge UI Style

See `.agentos/ui-style.md` for the canonical style contract.

Generated apps using this template must:

- Import `@agentos/ui/styles/agentos-forge.css`
- Use `@agentos/ui` components exclusively unless a missing component is explicitly required
- Default to `uiPreset: "agentos-forge"`
- Include command input, activity feed, run status, approval surface, quick actions, and generated app preview when relevant
